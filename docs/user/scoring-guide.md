# Scoring Guide

Lead Intake uses the hardcoded Decision Matrix. Staff do not edit scoring rules in the app; the form and score panel read from the matrix in code.

## Matrix Fields

- Client Type
- Budget Band
- Resource Consent
- Building Consent
- Building Stage
- Project Type
- Price-sensitivity Read
- Decision-makers
- Distance
- Source
- Payment History
- Site Access
- Installation Height

Unanswered fields stay unanswered. Do not guess a matrix answer just to improve completeness.

## Tiers

| Tier | Follow-up cadence |
|------|-------------------|
| A | Next day. |
| B | Within 7 days. |
| C | Within 21 days. |
| D | Light follow-up. |
| E | Nurture every 2-3 months or when project status changes. |

The score panel shows the current tier, score, completeness, last update, and follow-up date. The intake form automatically prefills the follow-up date unless the user has typed one manually.
