# Basic Staff Guideline

This is the short version for day-to-day use of **Lead Intake**, **Quote Tracker**, and **Work Order**.

## Menu Copy

| Menu item | Short description |
| --- | --- |
| Lead Intake | Record and score a new customer enquiry. |
| Quote Tracker | Create tracked quote links and check client engagement. |
| Work Order | View and manage active ServiceM8 work orders. |

## Quick Rule

Use the right tool for the job stage:

| Stage | Use |
| --- | --- |
| New enquiry | Lead Intake |
| Quote sent or ready to send | Quote Tracker |
| Job has moved to Work Order in ServiceM8 | Work Order |

## Lead Intake

Use **Lead Intake** when a new enquiry comes in by phone, email, WeChat, calculator, contact form, or another channel.

Basic steps:

1. Open **Lead Intake** from the menu.
2. Enter the customer details, job address, product, and job description.
3. Fill in the scoring fields you know.
4. Do not guess answers if the customer has not provided them.
5. Click **Save and score**.

Minimum details to save a lead:

- Client name or business name
- Phone
- Email
- Job address

Use Lead Intake to keep the enquiry clean before it becomes a quote or work order.

## Quote Tracker

Use **Quote Tracker** when the quote already exists in ServiceM8 and you want to send a tracked quote link to the customer.

Basic steps:

1. Make sure the quote PDF has been generated in ServiceM8.
2. Open **Quote Tracker** from the menu.
3. Click **Track Quote**.
4. Enter the ServiceM8 job number.
5. Copy the tracked quote link.
6. Send that link to the customer.
7. Check Quote Tracker later to see opens, viewing activity, and interest level.

Use the email gate before sending the link if you want the customer to enter their email before viewing the quote.

## Work Order

Use **Work Order** once the job has moved to **Work Order** status in ServiceM8.

Basic steps:

1. Open **Work Order** from the menu.
2. Refresh from ServiceM8 if the list needs updating.
3. Search or filter for the job.
4. Open the work order to view job details.
5. Update internal operational fields such as stage, installer, install date, risk, importance, or notes when needed.

Work Order is for managing active work, not for scoring new leads or creating quote links.

## Simple Handoff

- **Lead Intake** captures the enquiry.
- **Quote Tracker** helps send and follow up the quote.
- **Work Order** manages the job after it becomes active work.
